| 📊 仪表盘 | 📊 数据概览 | 更准确描述功能 |

**视频相关功能整合**:
- 📹 视频管理（父页面）

- **📊 数据概览**: 查看整体数据和管理视频
- **📝 内容优化**: 分析和优化视频内容（SEO、时长、发布时间、标签）

## 💡 设计理念

### 1. 按分析维度分组
- 数据维度：整体数据和管理
- 内容维度：视频内容相关分析
- 用户维度：用户和互动相关分析
- 系统维度：配置和设置

### 2. 父子关系整合
- 父页面：视频管理
- 子页面：单个视频详情、爆款提醒
- 使用缩进或图标表示层级关系

### 3. 工作流程导向
- 数据概览 → 内容优化 → 用户互动 → 系统设置
- 符合用户工作流程
- 减少不必要的跳转

### 4. 信息密度适中
- 不再过于分散
- 相关功能集中
- 避免单个分组过于庞大

## 🎨 实现方式建议

### 方案 A: 使用 expander（推荐）

```python
# 📊 数据概览
with st.sidebar.expander("📊 数据概览", expanded=True):
    if st.sidebar.button("📊 整体数据看板"):
        st.session_state.current_page = "overall_dashboard"
        st.rerun()
    
    if st.sidebar.button("📹 视频管理"):
        st.session_state.current_page = "video_management"
        st.rerun()
    
    # 子菜单
    with st.sidebar.expander("  └── 视频相关"):
        if st.sidebar.button("📹 单个视频详情"):
            st.session_state.current_page = "video_detail"
            st.rerun()
        if st.sidebar.button("🔥 爆款提醒"):
            st.session_state.current_page = "alerts"
            st.rerun()
```

### 方案 B: 使用多级按钮

```python
# 主分类
st.sidebar.markdown("#### 📊 数据概览")
if st.sidebar.button("📊 整体数据看板"):

    st.rerun()

    st.session_state.current_page = "video_management"
    st.rerun()

# 子菜单（缩进）

    st.session_state.current_page = "alerts"
    st.rerun()
```
