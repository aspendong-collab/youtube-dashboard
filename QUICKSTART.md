## 🎯 第一次使用

### 步骤 1：启动应用

```bash
streamlit run dashboard.py
```

### 步骤 2：配置 API 密钥

1. 打开 `http://localhost:8501`
2. 点击左侧侧边栏的 **"🔑 API 配置"**
3. 输入您的 YouTube Data API v3 密钥
4. 点击 **"保存 API 密钥"**

> **获取 API 密钥**：访问 [Google Cloud Console](https://console.cloud.google.com/)，创建项目并启用 YouTube Data API v3。

### 步骤 3：添加视频

1. 点击左侧侧边栏的 **"📹 视频管理"**
2. 输入 YouTube 视频 URL 或视频 ID
3. 点击 **"添加视频"**
4. 等待 1-2 秒获取数据

**示例 URL**：
- `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
- `https://youtu.be/dQw4w9WgXcQ`
- 或直接输入视频 ID：`dQw4w9WgXcQ`

### 步骤 4：查看数据分析

1. 点击 **"📊 整体看板"** 查看总体数据
2. 点击 **"📹 单个视频"** 深入分析特定视频
3. 点击 **"🎯 SEO 优化"** 获取优化建议

## 🔧 常见问题

### Q: 侧边栏没有显示更新后的样式？

**A**: 请确保：
1. 停止当前运行的应用（Ctrl+C）
2. 清除浏览器缓存
3. 重新启动应用：`streamlit run dashboard.py`

### Q: 如何获取 YouTube API 密钥？

**A**:
1. 访问 [Google Cloud Console](https://console.cloud.google.com/)
2. 创建新项目
3. 启用 "YouTube Data API v3"
4. 创建凭据 → API 密钥
5. 复制 API 密钥到应用中

### Q: 批量添加视频支持什么格式？

**A**: 支持 `.txt` 文件，每行一个：
```
https://www.youtube.com/watch?v=VIDEO_ID_1
https://www.youtube.com/watch?v=VIDEO_ID_2
VIDEO_ID_3
```

### Q: 数据保存在哪里？

**A**: 数据保存在 `youtube_dashboard.db` SQLite 数据库文件中。
